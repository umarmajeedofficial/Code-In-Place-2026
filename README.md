## 📌 Why This Repository Exists

Most learning repositories are just collections of code.  
This one is different.

This repo is built to act as a **learning system**, not just storage.

It captures:
- 📈 **Progress over time** (not just final solutions)
- 🧠 **Thinking process behind problems**
- 🔍 **Approach, mistakes, and improvements**
- 🧩 **Pattern recognition in problem solving**
- 🔁 **Consistency as a core skill**
